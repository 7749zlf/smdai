# 栈前日志

这是一个适合个人持续更新的前端知识博客。

## 直接打开

双击 [index.html](./index.html) 就能在浏览器里查看。

## 每天怎么发文章

1. 打开页面后，滚动到“写作面板”。
2. 填写日期、标题、标签、摘要和正文。
3. 点“发布文章”，新文章会立刻出现在博客里。

## 草稿说明

- 输入内容会自动保存在浏览器本地。
- 重新打开页面时，草稿会自动恢复。
- 你发布过的文章也会保存在浏览器本地。

## GitHub Pages

项目里已经加好了 GitHub Pages 工作流，文件在 [.github/workflows/pages.yml](./.github/workflows/pages.yml)。

上线步骤很简单：

1. 把这个项目推到 GitHub 仓库。
2. 去仓库 `Settings` -> `Pages`。
3. 把 `Source` 设成 `GitHub Actions`。
4. 推一次 `main` 分支，Pages 就会自动部署。

这套配置只会发布 `index.html`、`styles.css` 和 `script.js`，不会把工作区里那些无关文件一起带上去。
